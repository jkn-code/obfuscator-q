### Простой запутыватель скриптов проекта в годоте

-----

Заменяет на нечитаемые слова:
- объявленные через `var` переменные
- объявленные через `func` функции
- аргументы функций
- объявленные через `class_name` классы

Игнорирует все имена, которые начинаются не с маленькой английской буквы.

Переименует: `move_speed`, `push_button`, `cMonster`

Проигнорирует: `_ready`, `Player`

Еще игнорирует переменные, объявленные через `@export`.

-----

Использование:
- Скопировать папку проекта, назвать ее, например `/mygame-obf/`
- Скопировать в нее `obfuscator_q.exe` (из /Build) и запустить
- Указать нужные для игнорирования папки, файлы и имена (переменных, функций и классов)
- Нажать `Start`
- Программа заменит все найденные, и сохранит соответствия имен в `obfuscator_q_map.txt`
  
-----

Смысол:

Проект годот легко распаковывается для получения исходного кода.
Чтобы усложнить получение исходников другим человеком, его можно запутать с помощью `obfuscator-q`.
`Gdmaim` плохо работает с веб билдами, и в целом встречаются сложности.
