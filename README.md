Простой запутыватель скриптов проекта в годоте.

Заменяет на нечитаемые слова имена:
- объявленных через var переменных
- объявленных через func функций
- аргументы функций
- объявленных через class_name классов

Игнорирует все имена, которые начинаются не с маленькой английской буквы.
Игнорирует переменные, объявленные через @export.

Использование:
- Скопировать папку проекта, назвать ее, например /mygame-obf/
- Скопировать в нее obfuscator_q.exe и запустить
- Указать нужные для игнорирования папки, файлы и имена (переменных, функций и классов)
- Нажать Start
- Программа заменит все найденные

Смысол:
Gdmaim плохо работает с веб билдами, и в целом встречаются сложности.

Совет:
Внимательнее со списком игнорирования имен. 
Я вписал туда встречающиеся служебные поля, которые могут совпасть с объявленными переменными.
Но могут попасться еще.
